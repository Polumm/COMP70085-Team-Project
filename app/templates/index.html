Welcome to the Memory Game API

Below are the available API endpoints for this project:

  • Leaderboard:

    Fetch the top 10 player scores ranked by completion time.

    GET /leaderboard
  • Create Game:

    Generate a new random card layout and save it in the database.

    POST /create_game
  • Submit Score:

    Submit a player's game result (name, completion time, and moves).

    POST /submit_score
  • Get Card Layouts:

    Retrieve all saved card layouts in the database.

    GET /get_card_layouts
  • Get Random Images:

    Fetch a specified number of unique random images from an external API.

    GET /get_random_images?count=8
